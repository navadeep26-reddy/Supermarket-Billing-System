# Supermarket-Billing-System
The Supermarket Billing System is a simple Java console-based application designed to simulate a basic shopping and billing process in a supermarket.
## Features
- Pre-loaded with default products (Rice, Milk, Bread, Eggs, Oil)
- Add custom products with ID, name, and price
- Buy products by selecting ID and quantity
- Automatic calculation of total amount, GST (5%), and final bill
- Input validation for choices and quantities

## How to Run
1. Ensure Java is installed on your system.
2. Compile the program: `javac Main.java`
3. Run the program: `java Main`
4. Follow the on-screen menu to add products or make purchases.
5. Enter 0 to exit and view the bill.
